This is a mirror of [the tlnet directory](https://www.ctan.org/tex-archive/systems/texlive/tlnet) (minus docs, source, and some platform binaries) from CTAN to Cloudflare R2, taking advantage of Cloudflare's global cloud network for fast access worldwide. This is _not_ a full CTAN mirror, and only serves as a repository for [TeX Live](https://tug.org/texlive/) users (including [TinyTeX](https://yihui.org/tinytex/) users) to install / update packages via `tlmgr` after pointing `tlmgr option repository` to this site.

The site is maintained by [Yihui Xie](https://yihui.org), and automatically generated from the GitHub repo [yihui/ctan-tlnet](https://github.com/yihui/ctan-tlnet) on a daily basis.

TeX Live is developed as a joint effort by [all TeX user groups](https://tug.org/usergroups.html). This site does not modify any files from the tlnet directory, but only syncs a subset of files for TeX Live to function on major platforms. We do not claim copyright on any files hosted on this site.
